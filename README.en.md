# [RollSheet](https://ax4m3.github.io/RollSheet/)

A static, lightweight, and (almost) comprehensive web tool for generating, previewing, and exporting character sheets for various role-playing game systems (Dungeons & Dragons, Warhammer,...).

The project is written entirely in HTML, CSS, and pure JavaScript, and runs directly in the browser without requiring any installation.

## Features

- **Web Form**: Manually enter your character's name, avatar, stats, skills, and backstory.
- **Random Generation**: Create characters instantly using the *Random* button.
- **Multilingual system**: Interface available in Deutch (DE), English (EN), Español (ES), and Français (FR).
- **Responsive theme**: Built-in dark mode to protect your eyes during late-night sessions.
- **Export multi-formats** :
  - **📥 PDF**: Generate a print-ready character sheet via.
  - **📄 DOCX (Word)**: Export an editable file to fine-tune your character sheet in your word processor.
  - **💾 JSON**: Save your character's data structure to reload it later.
  - **Share**: Share your character sheet using the generated link provided.

## Use & Installation

### Option 1: Online Use (Recommended)
Go directly to the project page hosted by GitHub:
-> **[https://ax4m3.github.io/RollSheet/](https://ax4m3.github.io/RollSheet/)**

### Option 2: Local Use (Without Internet)
No server or installation (such as Python) is required.
1. Download or clone the repository :
   ```bash
   git clone [https://github.com/Ax4M3/RollSheet.git](https://github.com/Ax4M3/RollSheet.git)
   cd RollSheet
   ```
2. Double-click the `index.html` file to open the generator immediately in your browser.
