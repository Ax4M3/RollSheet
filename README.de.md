# [RollSheet](https://ax4m3.github.io/RollSheet/)

Ein statisches, ressourcenschonendes und (fast) vollständiges Web-Tool zum Erstellen, Anzeigen und Exportieren von Charakterbögen für verschiedene Rollenspielsysteme (Dungeons & Dragons, Warhammer,...).

Das Projekt ist vollständig in HTML, CSS und reinem JavaScript programmiert und läuft direkt im Browser, ohne dass eine Installation erforderlich ist.*

## Funktionen

- **Webformular**: Geben Sie die Identität, den Avatar, die Werte, die Fähigkeiten und die Geschichte Ihres Charakters manuell ein.
- **Zufallsgenerator**: Erstellen Sie Charaktere im Handumdrehen mit der Schaltfläche *Zufall*.
- **Mehrsprachiges System**: Die Benutzeroberfläche ist auf Deutsch (DE), English (EN),  Español (ES) et Français (FR) verfügbar.
- **Responsives Design**: Integrierte Verwaltung eines Dunkelmodus, um Ihre Augen bei nächtlichen Spielsitzungen zu schonen.
- **Export in verschiedene Formate** :
  - **📥 PDF**: Eine druckfertige datei im PDF-format erstellen.
  - **📄 DOCX (Word)**: Exportieren Sie eine bearbeitbare Datei, um Ihr Datenblatt in Ihrem Textverarbeitungsprogramm weiter zu verfeinern.
  - **💾 JSON**: Speichern Sie die Datenstruktur Ihres Charakters, um sie später wieder zu laden.
  - **Teilen**: Teilen Sie Ihre Charakterbeschreibung über den generierten Link.

## Bedienung und Installation

### Option 1: Online-Nutzung (empfohlen)
Rufen Sie direkt die von GitHub gehostete Projektseite auf:
-> **[https://ax4m3.github.io/RollSheet/](https://ax4m3.github.io/RollSheet/)**

### Option 2: Lokale Nutzung (ohne Internet)
Es sind weder ein Server noch eine Installation (wie Python) erforderlich.
1. Laden Sie das Repository herunter oder klonen Sie es:
   ```bash
   git clone [https://github.com/Ax4M3/RollSheet.git](https://github.com/Ax4M3/RollSheet.git)
   cd RollSheet
   ```
2. Doppelklicken Sie auf die Datei `index.html`, um den Generator sofort in Ihrem Browser zu öffnen.
